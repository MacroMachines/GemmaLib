<desc>;
draw a label to the display window.;

<args>;
x\, y = x and y coordinates in pixels (float);
p = name of preset to load (symbol);
s = size of text (float);
c = name of color to load (symbol);
d = degree of z-axis rotation (float);
r\, g\, b\, a = rgba values (int: 0-255);
f = the number of stored font to use (int: 1-10);
z = z-order(drawing order) (int: 1-100);
l = alignment of text (int: 0-2);
v = visible/invisible (int: 1/0);
t = text to draw (symbol);
